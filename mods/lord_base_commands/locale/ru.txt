### init.lua ###
chat action (eg. /me orders a pizza)	=	Опубликовать свой статус
Get help for commands or list privileges	=	Получить справку по командам или привилегиям
print out privileges of player	=	Вывести привилегии игрока
Give privilege to player	=	Дать привилегии игроку
Remove privilege from player	=	Забрать привилегии у игрока
set given password	=	Задать пароль
set empty password	=	Сбросить пароль
reload authentication data	=	Перезагрузить данные аутентификации
teleport to given position	=	Переместиться (или переместить другого игрока) на указанную позицию
set or read server configuration setting	=	Установить или вывести настройку конфигурации сервера
starts loading (or generating, if inexistent) map blocks contained in area pos1 to pos2	=	Начать загрузку (или генерацию, если не сгенерировано ранее) блоков от позиции pos1 до позиции pos2
delete map blocks contained in area pos1 to pos2	=	Удалить блоки от позиции pos1 до позиции pos2
List mods installed on the server	=	Вывести список модов, включённых на сервере
give item to player	=	Добавить предмет в инвентарь игрока
give item to yourself	=	Добавить предмет в свой инвентарь
Spawn entity at given (or your) position	=	Спавн моба на указанной позиции
Destroy item in hand	=	Уничтожить предмет, находящийся в руках
Check who has last touched a node or near it, max <seconds> ago	=	Вывести, кто последним изменял узел или узлы рядом с ним
revert actions of a player; default for <seconds> is 60	=	Откат действий игрока (по умолчанию на 60 секунд назад)
Print server status	=	Вывести статус сервера
set time of day	=	Вывести или установить игровое время суток
shutdown server	=	Выключить сервер
Ban IP of player	=	Добавить IP игрока в бан-лист
remove IP ban	=	Удалить IP из бан-листа
kick a player	=	Выкинуть игрока из игры
clear all objects in world	=	Очистить все объекты в мире
Send a private message	=	Отправить приватное сообщение
Get the last login time of a player	=	Вывести время последнего посещения игрока
